[https://developer.atlassian.com/cloud/jira/platform/rest/v3/intro/](https://developer.atlassian.com/cloud/jira/platform/rest/v3/intro/)

### 1. Issue APIs

| Method | Endpoint                                      | Brief Description              | Type   | Difficulty | Importance |
| ------ | --------------------------------------------- | ------------------------------ | ------ | ---------- | ---------- |
| GET    | /rest/api/3/issue/{issueIdOrKey}              | Get issue details by ID or key | Read   | Low        | High       |
| POST   | /rest/api/3/issue                             | Create an issue                | Create | Medium     | High       |
| PUT    | /rest/api/3/issue/{issueIdOrKey}              | Update an issue                | Update | Medium     | High       |
| DELETE | /rest/api/3/issue/{issueIdOrKey}              | Delete an issue                | Delete | Medium     | Medium     |
| GET    | /rest/api/3/issue/{issueIdOrKey}/comments     | List comments on an issue      | Read   | Low        | High       |
| POST   | /rest/api/3/issue/{issueIdOrKey}/comment      | Add comment to an issue        | Create | Medium     | High       |
| PUT    | /rest/api/3/issue/{issueIdOrKey}/comment/{id} | Update a comment               | Update | Medium     | Medium     |
| DELETE | /rest/api/3/issue/{issueIdOrKey}/comment/{id} | Delete a comment               | Delete | Medium     | Medium     |

[Docs](https://developer.atlassian.com/cloud/jira/platform/rest/v3/api-group-issues/)

### Summary

| API Group   | Typical Uses                | Complexity Range | Popularity |
| ----------- | --------------------------- | ---------------- | ---------- |
| Issues      | CRUD on issues/comments     | Low–Medium       | High       |
| Projects    | Manage projects             | Medium–High      | Medium     |
| Search      | Query issues                | Medium           | High       |
| Users       | User info and search        | Low              | High       |
| Issue Types | Metadata about issue types  | Low              | Medium     |
| Workflows   | Retrieve workflows          | Medium           | Medium     |
| Versions    | Manage versions             | Low–Medium       | Medium     |
| Components  | Manage components           | Low–Medium       | Medium     |
| Transitions | Manage issue status changes | Medium           | High       |
| Attachments | Add files to issues         | Medium           | Medium     |
| Issue Links | Metadata for linking issues | Low              | Medium     |
