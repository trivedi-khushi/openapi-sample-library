# Notion API Endpoint Catalog (2025)

Official docs:
[https://developers.notion.com/reference/intro](https://developers.notion.com/reference/intro)

---

### 1. Databases

| Method | Endpoint                     | Description                                      | Type | Difficulty | Importance |
| ------ | ---------------------------- | ------------------------------------------------ | ---- | ---------- | ---------- |
| POST   | /v1/databases/query          | Query a database with filters, sorts, pagination | Read | Medium     | High       |
| GET    | /v1/databases/{database\_id} | Retrieve a database’s metadata                   | Read | Low        | High       |

[Docs](https://developers.notion.com/reference/post-database-query)

---

### 2. Pages

| Method | Endpoint             | Description                              | Type   | Difficulty | Importance |
| ------ | -------------------- | ---------------------------------------- | ------ | ---------- | ---------- |
| POST   | /v1/pages            | Create a page in a database or workspace | Create | Medium     | High       |
| GET    | /v1/pages/{page\_id} | Retrieve a page’s properties             | Read   | Low        | High       |
| PATCH  | /v1/pages/{page\_id} | Update a page’s properties               | Update | Medium     | Medium     |

[Docs](https://developers.notion.com/reference/post-page)

---

### 3. Blocks

| Method | Endpoint                        | Description              | Type   | Difficulty | Importance |
| ------ | ------------------------------- | ------------------------ | ------ | ---------- | ---------- |
| GET    | /v1/blocks/{block\_id}          | Retrieve block data      | Read   | Low        | High       |
| PATCH  | /v1/blocks/{block\_id}          | Update a block           | Update | Medium     | Medium     |
| GET    | /v1/blocks/{block\_id}/children | Retrieve children blocks | Read   | Medium     | High       |
| PATCH  | /v1/blocks/{block\_id}/children | Append children blocks   | Create | Medium     | High       |

[Docs](https://developers.notion.com/reference/get-block)

---

### 4. Users

| Method | Endpoint             | Description                      | Type | Difficulty | Importance |
| ------ | -------------------- | -------------------------------- | ---- | ---------- | ---------- |
| GET    | /v1/users            | List users                       | Read | Low        | Medium     |
| GET    | /v1/users/{user\_id} | Retrieve a user by ID            | Read | Low        | Medium     |
| GET    | /v1/users/me         | Retrieve current authorized user | Read | Low        | High       |

[Docs](https://developers.notion.com/reference/get-users)

---

### 5. Search

| Method | Endpoint   | Description                                | Type | Difficulty | Importance |
| ------ | ---------- | ------------------------------------------ | ---- | ---------- | ---------- |
| POST   | /v1/search | Search across pages, databases, workspaces | Read | Medium     | High       |

[Docs](https://developers.notion.com/reference/post-search)

---

### Summary

| API Group | Use Cases                          | Difficulty Range | Popularity/Importance |
| --------- | ---------------------------------- | ---------------- | --------------------- |
| Databases | Query database contents            | Medium           | High                  |
| Pages     | Create, retrieve, update pages     | Low–Medium       | High                  |
| Blocks    | Manage content blocks within pages | Low–Medium       | High                  |
| Users     | Retrieve user info                 | Low              | Medium                |
| Search    | Search across workspace            | Medium           | High                  |
